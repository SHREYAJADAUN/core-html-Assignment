HTML TAGS

  1. Paragraph tag

    2. This HTML element represents a paragraph. Paragraphs are block-level elements.


  2. Title tag

    3. The HTML element defines the document's title that is shown in a browser's title bar or a page's tab. It only contains text; tags within the element are ignored.


  3. Line break element

    4. This HTML element produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.

  4. Thematic break : Horizontal rule

    5. This HTML element represents a thematic break between paragraph-level elements: for example, a change of scene in a story, or a shift of topic within a section.

  5. Anchor tag

    6. This HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files,
    email addresse assocations in the same page, or anything else a URL can address.


  6. Image tag

    7. The HTML element embeds an image into the document.
